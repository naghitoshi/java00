## クラス、メソッド、フィールド
- クラス：特定の目的を達成するのに必要なものをまとめたもの
- メソッド：クラス内で特定の処理を行うために必要なプログラムをまとめたもの。関数に近い概念。
- フィールド：クラスの中でデータの値を保存するために使用するもの。メンバ変数に近い概念。
```
class クラス{
    データ型 フィールド名; //int A = 0;など

    void メソッド(){
        //
        }
}
```
